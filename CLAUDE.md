# Contexto del proyecto

TP obligatorio de Programación 2 en Java.

Alternativa elegida: A - Ecosistema de Red Social Profesional.

Requisitos:
- Buscar usuarios por ID en tiempo promedio O(1), usando HashMap.
- Representar conexiones profesionales con un grafo.
- Calcular grado de separación usando BFS.
- Sugerir contactos recomendados según vínculos comunes.
- Organizar habilidades en forma jerárquica con un árbol.
- Procesar postulaciones en orden de llegada usando cola FIFO.
- Permitir deshacer la última actualización del perfil usando pila LIFO.
- Código modular, legible, con validaciones y casos de prueba.
- Usar Git con commits separados y descriptivos.

Reglas del desarrollo:
- Usar Java sin frameworks externos.
- Código simple y entendible para nivel Programación 2.
- Separar las clases por responsabilidad.
- Agregar casos de prueba o una clase Main que demuestre cada funcionalidad.
- Hacer commits pequeños y descriptivos.
